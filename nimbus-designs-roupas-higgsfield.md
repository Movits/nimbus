# NIMBUS — Estampas por coleção

A loja é organizada em **coleções**. Cada coleção tem sua cara (bloco de estilo) e seus designs.
Cada prompt é copy-paste e **autossuficiente** (o Higgsfield não tem memória) — já traz o estilo inteiro.

## Como gerar
- Modelo **Nano Banana 2** (ilimitado) · **2K** · aspect ratio indicado · anexe **ref** só onde indicado.
- **Fundo VERDE sólido `#00B140`** — EXCETO **PADROEIRA**, que gera em **MAGENTA `#FF00FF`** (a arte
  dela tem verde, que brigaria com o chroma verde).
- Salve em `designs/originais/` com o **código no nome** (ex.: `Y1-nimbus-chrome.png`).
- Rode `node scripts/organize-designs.mjs` → recorta o fundo (auto-detecta verde/magenta/xadrez),
  300 DPI, e organiza em `designs/prontos/<COLECAO>/{costas,peito}`.
- Cores das peças: **preto** (arte clara/colorida) + **off-white** (arte escura). Auditoria de
  colorway (quais precisam da versão invertida) fica pra depois do teste na YouDraw.

## Pastas + workflow
```
designs/
  originais/                       <- solte as gerações aqui (verde/magenta), com o código no nome
  prontos/
    STREET/{costas,peito,mockups}     <- G*
    RELIQUIA/{costas,peito,mockups}   <- B*, H*
    GLORIA/{costas,peito,mockups}     <- Y*
    PADROEIRA/{costas,peito,mockups}  <- S*  (geradas no magenta)
    NUVEM/{costas,peito,mockups}      <- artes antigas de nuvem
    _marca/                           <- logo (peito em qualquer coleção)
```
Gerar (Nano Banana 2 + fundo certo + aspect ratio, ref só onde indicado) → `designs/originais/`
→ `node scripts/organize-designs.mjs` → recorta + 300 DPI + organiza por coleção. Mockups vão em
`prontos/<COLECAO>/mockups/` (skill higgsfield-product-photoshoot).

> Imagens de design **não vão pro GitHub** (repo público) — ficam local + Drive. As artes antigas
> de emblema-nuvem estão em `prontos/NUVEM/` (disponíveis pra um mix).
